A simple and responsive Student Management System built using HTML, CSS, and JavaScript. This project allows users to add, edit, delete, and search student records with data stored in the browser using localStorage.

Features

 * Add new student details

* Edit existing student records

* Delete student data

* Search students by Register Number, Name, Email, or Course

* Data stored in localStorage (persists after page refresh)

* Stylish and modern UI design

* Responsive layout

Technologies Used

* HTML5

* CSS3

* JavaScript (Vanilla JS)

 * Browser LocalStorage
